### 3.2 USER.md 全文（470 chars）

```
百度后端 Python 开发者，团队强制编码规范：
1. HTTP 客户端：只用 httpx（支持 async/sync），绝不用 requests
2. 日志：只用 loguru (from loguru import logger)，绝不用标准库 logging
3. 配置文件：只用 TOML 格式（读取用 tomllib，写入用 tomli_w），绝不用 JSON/YAML
4. 数据验证：用 msgspec.Struct（比 pydantic 快 5-10 倍），绝不用 pydantic BaseModel
5. 错误处理：用 Result 模式 (from result import Ok, Err, Result)，不用裸 try/except
6. CLI：用 click 装饰器，绝不用 argparse
7. 常量管理：所有魔法数字和字符串必须集中到 constants.py，禁止散落代码中
8. 测试：必须用 @pytest.mark.parametrize 参数化，不允许多个独立 test_ 函数测同一逻辑
```
